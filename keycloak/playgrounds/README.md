# Keycloak Playgrounds

## English
This directory contains standalone projects for exploring and experimenting with specific Keycloak features.

These playgrounds are **not part of any training course**. They serve as a sandbox for trying out functionality that may be incorporated into future training materials.

### Available Playgrounds

| Directory | Topic |
|-----------|-------|
| [`kerberos/`](kerberos/) | Keycloak integration with Kerberos/SPNEGO authentication |
