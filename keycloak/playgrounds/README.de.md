# Keycloak Playgrounds

Dieses Verzeichnis enthält eigenständige Projekte zum Erkunden und Ausprobieren spezifischer Keycloak-Funktionen.

Diese Playgrounds sind **kein Bestandteil eines Trainingskurses**. Sie dienen als Experimentierfläche für Funktionen, die möglicherweise in zukünftige Trainingsmaterialien einfließen.
